## EstacolNews

Software encargado de administrar articulos o noticias de los escritores mas famosos actualmente, los editores pueden crear, eliminar, publicar, editar y demas funciones acerca de sus creaciones, tambien podemos ser espectadores por ende podemos compartir noticias o articulos con nuestros amigos atravez de redes sociales como Linkedin o Whatsapp, ademas de eso podemos reaccionar a las diferentes noticias que nos puedan interesar y dejar nuestro "LIKE".
